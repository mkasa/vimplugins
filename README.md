# vimplugins
My vim plugins (in addition to Janus)

## How to use

```
% cd
% git --version
(make sure that you have git 1.6.5 or later)
% git clone --recursive https://github.com/mkasa/vimplugins.git .janus
% cd .janus
% make init
% make link
```
